# Dossiê final

## Instalação do Projeto

Clone o repositório e instale as dependências.

```bash
# Clonar o repositório
git clone https://github.com/eded001/dossie-final.git

# Entrar na pasta do projeto
cd dossie-final

# Instalar dependências
npm install
```

---

## Rodando a Aplicação

Após instalar tudo, execute:

```bash
npm run dev
```

---

## Acessando no Navegador

Com o servidor rodando, abra o navegador e acesse:

```text
http://localhost:3000
```

A aplicação estará disponível nessa porta em ambiente de desenvolvimento.

---

## Para parar o servidor

No terminal onde está rodando:

```text
CTRL + C
```
